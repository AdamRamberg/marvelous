## Installation 
Marvelous can be installed through npm's registry. Add the following to your `manifest.json`:
```
{
    "scopedRegistries": [
        {
            "name": "Marvelous",
            "url": "https://registry.npmjs.org",
            "scopes": [
                "com.mambojambostudios.marvelous"
            ]
        }
    ],
    "dependencies": {
        ...
        "com.mambojambostudios.marvelous" : "1.0.2",
        ...
    }
}
```
